## RFC Format

Each RFC includes:

1. **Summary** - One paragraph overview
1. **Motivation** - Why this is needed
1. **Design** - How it works
1. **Implementation** - Where to find the code
1. **Drawbacks** - Known limitations
1. **Alternatives** - Other approaches considered
